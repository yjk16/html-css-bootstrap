### 2. Viewing Instructions

1. Click on the green `code` button and select `download ZIP`
2. Extract the downloaded file
3. Navigate to the desired file and open it in a browser to view:
    
    - `intro/index.html` for the simple HTML + CSS page
    - `shopping app/shopping_page.html` for the HTML + CSS + additional Bootstrap styling
